01 · Premise
The problem

Promises are soft.
Stakes are not.

Every relationship runs on promises — the unspoken kind and the written kind. They erode quietly. Nobody signs them. Nobody enforces them.

When they break, both parties debate what was ever agreed in the first place.

03 / 08
Thesis
The goal is not punishment. It is legibility — of what was promised, and what was kept.
— LoveClaw, design principle no. 1

02 · Architecture
How it's built

Four layers, one commitment.

01 · client

The local agent.

Privacy-first daemon on each phone. Observes only the signals both partners agreed to share. Raw data never leaves the device.

02 · transport

The secure relay.

End-to-end encrypted channel couriering signed messages. Sees timing and size — never content.

03 · logic

The rule engine.

Partners co-author rules that compile to predicates. Evaluation is mutual — both agents must agree.

04 · settlement

The smart contract.

Shared vault holds collateral. Breach routes funds per pre-agreed terms. Graduation returns it in full.

03 · Lifecycle
From signal to settlement

One breach, in five beats.

T+00s

Signal observed

Local agent on Phone A registers an event matching a monitored predicate.

T+02s

Cross-check

Encrypted request to Phone B's agent confirms independent context.

T+04s

Rule triggers

Predicate evaluates true on both sides. Breach state is ratified.

T+06s

Contract called

Signed attestation submitted on-chain. A grace window opens.

T+72h

Settlement

If uncontested, collateral routes per agreement. Both partners notified.

04 · Principles
Non-negotiable

What LoveClaw refuses to do.

The protocol is opinionated. It exists to make commitments legible — not to surveil, coerce, or punish.

001 · Consent

No rule without both signatures.

Every predicate is co-authored and co-signed. Either partner can veto. Unilateral monitoring is not a feature.

002 · Locality

Raw signals never leave the device.

Only attestations cross the wire. Location, messages, and app contents remain on-device at all times.

003 · Exit

Graceful dissolution is first-class.

Ending the protocol returns all collateral with no penalty. Leaving a relationship is never a breach.
